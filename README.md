# Operator Repo

Control hub for two NanoClaw bots running on Pi5:

- **Andy** at `~/nanoclaw-andy/` — Kyle's assistant. Discord: `NanoClaw-Andy`.
- **Milton** at `~/nanoclaw-milton/` — Alexa's paralegal. Discord: `NanoClaw-Milton`.

This repo holds operator instructions, an overlay reference for re-applying customizations after upstream pulls, and helper scripts. **It is not a NanoClaw install** — see `CLAUDE.md` for the full operator playbook.

```
operator/         helper scripts (status checks, etc.)
overlay/          source files / patches to re-apply after upstream pulls
CLAUDE.md         operator playbook (Claude Code reads this)
README.md         this file
```
